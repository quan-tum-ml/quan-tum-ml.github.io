    Ubi loqui #

    The task of learning an arbitrary function from data is mathematically expressed as the minimization of a loss function \(L(\bm{\theta})\) , also known as the objective function, with respect to the parameter vector \(\bm{\theta}\)

    For the loss function, a standart quadratic loss was used:

    Square loss

    \(L(\bm{\theta}) = (y - ≈∑)^{2}\)

    Where y is the ground truth label and ≈∑ is the predicted label

    Note that this only works since we are using the confidence multiplied by its label and not the binary label itself. If we wished to do that, we would have to use a different loss function, e.g. binary cross entropy loss.

    How does one get the data into the Quantum circuit? #

    There are a multitude of ways to do so. The two simplest ways to embed information are Amplitude and Product encoding.

    Amplitude encoding #

    Amplitude encoding only works if the input features x are nomalized to \(\bm{x} \in [0,1] \) . Then the Quantum states are prepared as following:

    Amplitude encoding

    \( x \in\{0,1\}^{n} \rightarrow |x \rangle \)

    This of cause also works when working with binary input features, then it is called Basis encoding Amplitude embedding gives rise to a linear kernel in feature Hilbert space 3 and while it is consideres memory efficient, unfortunately ,the depth of this encoder circuit is expected to scale ex-ponentially with the number of qubits for generic inputs.Therefore, algorithms based on amplitude encoding couldbe impeded by our inability to coherently load data intoquantum states. 2
